# Agroguid-Ecosistem
Экосистема АгроГид предназначена для помощи подросткам в выборе будущей профессии в сфере АПК, а также экосистема помогает распространить профессии в этой сфере. Она представляет собой структуру Телеграм-бот + сайт. Бот предназначен для профориентации и выбора интересующего направления в этой сфере. Сайт же рассказывает о каждом направлении.
